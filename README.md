# Project Progress  
## โครงงานระบบ Cloud Storage ส่วนตัวด้วย Raspberry Pi และ Nextcloud

---

## 1. หลักการและเหตุผลของโครงงาน

ปัจจุบันบริการ Cloud Storage เช่น Google Drive, OneDrive หรือ Dropbox
ได้รับความนิยมอย่างแพร่หลาย แต่มีข้อจำกัดด้านความเป็นส่วนตัว ค่าใช้จ่าย และการควบคุมข้อมูล

โครงงานนี้มีแนวคิดในการสร้างระบบ Cloud Storage ส่วนตัว
โดยใช้ Raspberry Pi เป็น Server และใช้ซอฟต์แวร์ Nextcloud
เพื่อให้ผู้ใช้งานสามารถจัดเก็บ แบ่งปัน และเข้าถึงไฟล์ผ่านเครือข่ายได้
โดยเจ้าของระบบสามารถควบคุมข้อมูลได้ทั้งหมด

## 2. วัตถุประสงค์ของโครงงาน

1. เพื่อศึกษาและใช้งาน Raspberry Pi ในบทบาทของ Server
2. เพื่อสร้างระบบ Cloud Storage ส่วนตัวแบบ Multi-user
3. เพื่อเรียนรู้การทำงานของ Nextcloud และ Web Server
4. เพื่อศึกษาการจัดการสิทธิ์ผู้ใช้งานและข้อมูลบนระบบ Linux
5. เพื่อนำความรู้ด้านระบบเครือข่ายและ Server มาประยุกต์ใช้งานจริง

## 4. ตัวอย่างการทำงานของระบบ (Video)

วิดีโอแสดงตัวอย่างการทำงานเบื้องต้นประกอบด้วย
1. การเข้าสู่ระบบ (Login)
2. การสร้างผู้ใช้ใหม่โดย Admin
3. การอัปโหลดไฟล์
4. การแยกพื้นที่จัดเก็บของผู้ใช้แต่ละคน
5. การเข้าถึงระบบผ่าน IP Address

## 5. สรุปสิ่งที่ทำเสร็จสิ้นแล้ว

- ติดตั้ง Raspberry Pi OS (Desktop)
- ตั้งค่าเครือข่ายและ IP Address
- ติดตั้ง Apache Web Server
- ติดตั้ง PHP และส่วนเสริมที่จำเป็น
- ติดตั้ง MariaDB
- ติดตั้ง Nextcloud
- เชื่อมต่อ External Storage
- สร้างผู้ใช้งานหลายบัญชี (Multi-user)
- ทดสอบการอัปโหลดและแยกไฟล์ผู้ใช้
- ตรวจสอบโครงสร้างไฟล์บนระบบปฏิบัติการ

---

## 6. แผนการดำเนินงานในขั้นต่อไป

1. ตั้งค่า HTTPS ด้วย SSL Certificate
2. ปรับปรุงความปลอดภัยของระบบ
3. ตั้งค่า Backup ข้อมูลอัตโนมัติ
4. ศึกษาการเข้าถึงจากภายนอกเครือข่าย (Port Forwarding / VPN)
5. ปรับแต่งประสิทธิภาพระบบสำหรับผู้ใช้หลายคน
6. จัดทำเอกสารและรายงานฉบับสมบูรณ์

---

## บทบาทของ Raspberry Pi ในโครงงาน

Raspberry Pi ทำหน้าที่เป็นศูนย์กลางของระบบ (Server)
ซึ่งทำหน้าที่ประมวลผลคำขอจากผู้ใช้งาน
จัดการข้อมูล และควบคุมการเข้าถึงไฟล์
เปรียบเสมือน Cloud Server ขนาดเล็กที่สามารถควบคุมได้ด้วยตนเอง
